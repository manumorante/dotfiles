# dotfiles

Personal configuration files for development environment.

## Contents

- **[git](git/)** - Git configuration with conditional personal/work settings
- **[hushlogin](hushlogin/)** - Suppress terminal login messages
- **[iTerm2](iTerm2/)** - iTerm2 terminal profile configuration
- **[oh-my-zsh](oh-my-zsh/)** - Custom aliases and functions for zsh
- **[prettier](prettier/)** - Code formatter configuration
- **[zsh](zsh/)** - Main zsh configuration (.zshrc)
